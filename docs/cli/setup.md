# `omniclaw setup`

Initialize `~/.omniclaw/omniclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
omniclaw setup
omniclaw setup --workspace ~/.omniclaw/workspace
```

To run the wizard via setup:

```bash
omniclaw setup --wizard
```
