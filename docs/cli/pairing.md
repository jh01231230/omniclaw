# `omniclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
omniclaw pairing list whatsapp
omniclaw pairing approve whatsapp <code> --notify
```
