## SafeClaw Intent Integration

### Created

- `skills/intent_tracker/safeclaw_integration.py` - Bridge module for SafeClaw integration

### What It Does

- Detects project intents ("我想做个XX") → creates 4-phase project plan
- Detects progress updates ("完成了") → advances project phase
- Detects habit intents ("我想健身") → creates habit tracking
- Generates natural follow-ups on casual messages
